# gold-bot — шпаргалка

Локальный код: `~/gold-bot`. ХОСТИНГ: GitHub Actions (облако), ноут не нужен.
Репо: github.com/adilzhan2/gold-bot (ЛИЧНЫЙ аккаунт, публичный).

## Где работает (с 2026-06-16)
Бот крутится в GitHub Actions — облако само запускает опрос каждые 5 мин.
Ноут можно выключать. Локальный LaunchAgent ОТКЛЮЧЁН (файл .plist.disabled).

## Проверить, что бот жив
- Открой github.com/adilzhan2/gold-bot → вкладка **Actions** → workflow
  "gold-bot alerter". Зелёные галки = работает, каждые ~5 мин новый прогон.
- Или в терминале:
```bash
cd ~/gold-bot && gh run list --repo adilzhan2/gold-bot --workflow alerter.yml --limit 5
```

## Пауза / возобновить
```bash
gh workflow disable alerter.yml --repo adilzhan2/gold-bot   # пауза
gh workflow enable  alerter.yml --repo adilzhan2/gold-bot   # снова
```
(или на вкладке Actions → "..." → Disable/Enable workflow)

## Изменить код / стратегию
Правишь локально в ~/gold-bot → `git push` → следующий прогон уже на новом коде.
Секреты (токены) лежат в Settings → Secrets репозитория, не в коде.

## Удалить пустой репо с РАБОЧЕГО аккаунта (когда руки дойдут)
github.com/AdilzhanAkhayev/gold-bot → Settings → низ → Delete repository.

## Что делает бот
Раз в 3 минуты смотрит золото (спот XAU/USD через Twelve Data), ищет сетапы по
моему чек-листу (зона → свип → CHoCH → имбаланс), оценивает моделью «взял бы я
или нет» и шлёт в Telegram сигнал (ПОКУПАЙ/ПРОДАВАЙ + SL/TP в $) с вердиктом.
Все сетапы + картинки пишутся в `~/gold-bot/data/alerts/`.

⚠️ Это ПОДСКАЗКА, не сигнал на слепой вход. Появилось уведомление — смотрю
график сам по чек-листу.

Фид: Twelve Data (data/twelvedata.json). Сменить ключ:
  python -m scripts.twelvedata_setup <КЛЮЧ>
Лимит free: 800 запросов/день, опрос раз в 3 мин = ~480/день, ок.

## Журнал сделок (сверка модель vs реальность)
```bash
cd ~/gold-bot && source .venv/bin/activate
python -m scripts.journal             # список алертов + статистика
python -m scripts.journal 3 loss -2.46  # алерт №3 — стоп, минус $
python -m scripts.journal 3 win 35.9    # алерт №3 — тейк, плюс $
python -m scripts.journal 3 skip        # я пропустил этот
```
Цель: набрать 10-15 записей, увидеть — у ✅-сетапов winrate выше ⚠️ или нет.
Риск на сделку держать 1-2% депозита (не 12% как на старте).
