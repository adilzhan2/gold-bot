# gold-bot — шпаргалка

Проект: `~/gold-bot` (был на Desktop, перенесён — macOS не даёт автозапуск из Desktop).

## Автозапуск
Алертер сам стартует при входе в систему (LaunchAgent). После перезагрузки
Mac ничего делать не надо — он уже работает.

## Проверить, что бот жив
```bash
pgrep -fl scripts.alerter            # есть строка = работает
launchctl list | grep goldbot        # 2-я колонка 0 = ок
tail -f ~/gold-bot/data/alerts/alerter.log   # живой лог (Ctrl+C выйти)
```
В логе строки вида `... ok: 4219 баров (GC=F), свежих: 0, отправлено: N`.
`отправлено: 1+` = было уведомление.

## Остановить / запустить вручную
```bash
launchctl unload ~/Library/LaunchAgents/com.adilzhan.goldbot.alerter.plist   # стоп
launchctl load   ~/Library/LaunchAgents/com.adilzhan.goldbot.alerter.plist   # старт
```

## Что делает бот
Раз в 3 минуты смотрит золото (спот XAU/USD через Twelve Data), ищет сетапы по
моему чек-листу (зона → свип → CHoCH → имбаланс), оценивает моделью «взял бы я
или нет» и шлёт в Telegram сигнал (ПОКУПАЙ/ПРОДАВАЙ + SL/TP в $) с вердиктом.
Все сетапы + картинки пишутся в `~/gold-bot/data/alerts/`.

⚠️ Это ПОДСКАЗКА, не сигнал на слепой вход. Появилось уведомление — смотрю
график сам по чек-листу.

Фид: Twelve Data (data/twelvedata.json). Сменить ключ:
  python -m scripts.twelvedata_setup <КЛЮЧ>
Лимит free: 800 запросов/день, опрос раз в 3 мин = ~480/день, ок.

## Журнал сделок (сверка модель vs реальность)
```bash
cd ~/gold-bot && source .venv/bin/activate
python -m scripts.journal             # список алертов + статистика
python -m scripts.journal 3 loss -2.46  # алерт №3 — стоп, минус $
python -m scripts.journal 3 win 35.9    # алерт №3 — тейк, плюс $
python -m scripts.journal 3 skip        # я пропустил этот
```
Цель: набрать 10-15 записей, увидеть — у ✅-сетапов winrate выше ⚠️ или нет.
Риск на сделку держать 1-2% депозита (не 12% как на старте).
